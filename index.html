Find information about your favorite artist here. Find albums, songs, and other relevant information. Check out our features below to begin using Project Red.

Info

What is Project Red?
Spotify Aggregator that allows an individual to cycle through albums and songs of an Artist
How do I use the feature.
  1. First access the "Feature Panel".
  2. Enter an artist you would like to view.
  3. Select through available albums to view related information.
What is this currently built with?
HTML & CSS
How do you plan to scale this?
The future will see the usage of the Spotify API to query data, and the usage of NodeJS and React to organize and pass through information.
